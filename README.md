# 🔐 C++ File Encryption Tool

<p align="center">
<img src="https://capsule-render.vercel.app/api?type=waving&color=0:0f2027,50:203a43,100:2c5364&height=120&section=footer"/>
</p>

<p align="center">
<img src="https://raw.githubusercontent.com/devicons/devicon/master/icons/cplusplus/cplusplus-original.svg" width="80">
</p>

<p align="center">
A <b>console-based file encryption and decryption tool</b> written in <b>C++</b>.  
This program uses a simple <b>Caesar Cipher algorithm</b> to encrypt and decrypt text files while demonstrating file handling and string manipulation concepts.
</p>

---

# ✨ Features

🔐 **Encrypt Text Files**
🔓 **Decrypt Encrypted Files**
📂 **Read Data From Input Files**
💾 **Save Results Into Output Files**
🖥️ **Displays Encryption / Decryption in Terminal**
🧩 **Menu Driven CLI Program**

---

# 🧠 Concepts Demonstrated

| Concept             | Implementation         |
| ------------------- | ---------------------- |
| File Handling       | `ifstream`, `ofstream` |
| String Manipulation | Character shifting     |
| Encryption          | Caesar Cipher          |
| Control Flow        | `switch` menu system   |
| Loops               | File line processing   |
